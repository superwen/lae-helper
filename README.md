Helpers for laravel-admin
=========================


## Installation

```

$ composer require "superwen/helpers"
```

Import menu items.

```shell
$ php artisan admin:import helpers
```

## Usage

See [wiki](http://laravel-admin.org/docs/#/en/extension-helpers?id=helpers)
